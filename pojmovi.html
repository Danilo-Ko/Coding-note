토큰(token,token )은 네트웍에서 말하는 토큰이란, 토큰링 네트웍을 따라 돌아다니는 일련의 특별한 비트열이다. 컴퓨터들은 네트웍을 따라 순환하는 토큰을 자신이 잡았을 때만 네트웍에 메시지를 보낼 수 있다. 각 네트웍에는 오직 단 한 개의 토큰만이 존재함으로써, 두 개 이상의 컴퓨터가 동시에 메시지를 전송할 가능성을 사전에 차단한다.

Token u računarstvu je objekat u softveru ili hardveru koji predstavlja pravo na realizaciju neke operacije, >najčešće u računarskim mrežama pri razmeni podataka između stanica mreže.

2진법(vinarni operator), 10진법(Dekarni operator), 16진법(Heksadecimalni operator)

보수는 보충을 해주는 수를 의미합니다. 현재의 숫자에서 자릿수가 1단계 올라갈 수 있는 수가 몇인지를 구하시면 됩니다. 컴퓨터에서 보수는 양수를 음수화로 표현하기 위한 방법이 됩니다.

보수

10진수 결과값이 양수일때는 이진수에서 4비트 최상위 값을 제일 밑으로 내려주어야 하고 10진수 결과값이 음수일때는 이진수에서 4비트로 나온 값을 한 번 더 반전한 다음에 -를 적어주어야 한다.
document.write(~z); // 1의 보수와 같으므로 -(15+1) = -16 // 15는 1111 여기서 (~) 반전 즉 1의 보수를 해주면 10000이고 이 값이 바로 5 비트에서 16이다.

VS code 단축키 모음

shift + alt + ↓, ↑: 해당 줄을 위 혹은 아래로 복사
alt + ↓, ↑: 줄을 차례대로 위 혹은 아래로 선택
Ctrl + 좌클릭: 코딩할 때 커서를 해당되는 곳에 동시에 놓고 추가 밎 삭제 가능
shift + delete: 해당 줄을 삭제.
ctrl + shift + 엔터: 위에 줄 하나 삽입.
ctrl + enter: 아래에 줄 하나 삽입.
ctrl + (+,-): 화면 크기를 크거나(+) 작게(-) 함
ctrl + shift: 먼저 ctrl을 누른 상태에서 로 원하는 곳에 클릭한 후, 다음 ctrl과 shift를 동시에 누르고 (오른쪽 화살표)드래그 한 후 내용 변경이나 붙여넣기하면 됨
ctrl + shift + L: 같은 단어를 찾아주어 삭제하거나 변경할 수 있도록 해 줌. shift + alt: 아래로 마우스를 드래그하면 드래그 한 만큼 한 꺼번에 그 줄들에 커서가 생김. 한꺼번에 입력하거나 지울때 아주 유용
